# Setup
Para acessar esse projeto na sua máquina, basta clonar esse repositório:<br />
```
git clone https://github.com/naccaratocarolina/marvel-invaders-js.git
```
E, em seguida, acessar o arquivo ```index.html``` no seu browser.
